-- LittleRedSonja Cosmetic Armor System (CAS) -- 7 days to die V2 mod -- by LittleRedSonja --

Installation Instructions:

Extract the downloaded zip into your 7D2D Mods folder (create one if it does not exist).

To verify you have installed the mod correctly, the ModInfo.xml file should be located at Mods\LittleRedSonja_CAS\ModInfo.xml relative to your 7D2D install folder.


This mod IS NOT Server-Side. Since it adds custom c# code it requires installation both in server AND client for multiplayer. For solo player install like any other mod.


For this mod to work EAC must be OFF


ALWAYS DELETE THE PREVIOUS VERSION OF THE MOD WHEN UPDATING TO A NEW VERSION. DO NOT INSTALL ON TOP OF AN OLDER VERSION. 


Let me know if you want me to add support for other custom armors or outfits. 


I hope you enjoy the mod 💜. Let me know too if you have suggestions or find bugs.  🥰


HOW TO USE THE CONFIGURATION FILE:

1- Open it with any text editor (for example Notepad)
2- You will see an explanation of the 3 options, and this line at the bottom of the file: cosmetic_mode: UnlockAll
3- Replace the value there (UnlockAll by default) with the option you like best (Standard, UnlockAll, Disabled). You have some images for visual help in a folder, in the mod folder, and also in the Nexus page for the mod.
4- Save the file
5- Enjoy and have fun :)
